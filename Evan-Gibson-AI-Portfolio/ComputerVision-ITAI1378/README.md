# Computer Vision — ITAI 1378

Houston Community College | Applied AI & Robotics Program

## Course Overview
Fundamentals of computer vision including image processing, object detection, classification, and model evaluation. Projects use Google Colab.

## Projects

### Labs 1-11: Class Work and Teaching
### [Face Mask Detection Classifier](Face-Mask-Detection/)
Fine-tuned ResNet-18 to classify face mask wear status (correct, incorrect, none) across ~9,000 images. Achieved 99.6% test accuracy. Includes real-world inference testing and honest failure case analysis.
